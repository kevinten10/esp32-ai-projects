# ESP32 气象站 - PC 模拟器

无需开发板，直接在电脑上运行和测试！

## 🚀 快速启动

### 方法 1: 双击运行
```
simulator/run.bat
```

### 方法 2: 命令行
```bash
cd simulator
python weather_station.py
```

## 📋 功能说明

### GUI 界面
- 🌡️ **实时温度显示** - 模拟 DHT22 传感器数据
- 💧 **实时湿度显示** - 自动波动更新
- 🔥 **体感温度计算** - 根据温湿度计算
- ⏰ **时间显示** - 系统时间

### 交互控制
- 🌡️ **升温/降温** - 调整基础温度
- 💧 **加湿/除湿** - 调整基础湿度
- 🌐 **Web 服务器** - 启动本地 Web 服务
- 🔗 **打开浏览器** - 访问 Web 界面

### Web 界面
启动 Web 服务器后，访问：
- **本地访问**: http://localhost:8080
- **局域网访问**: http://你的 IP:8080

功能：
- 实时数据显示
- 自动刷新（5 秒）
- JSON API: `/data`
- 互动控制（升温/加湿）

## 🎮 使用示例

1. **启动模拟器**
   ```bash
   run.bat
   ```

2. **调整环境参数**
   - 点击 "升温" 按钮，温度上升
   - 点击 "加湿" 按钮，湿度上升

3. **启动 Web 服务**
   - 点击 "启动 Web 服务器"
   - 点击 "打开浏览器"

4. **查看数据**
   - GUI 界面实时显示
   - Web 界面远程访问
   - JSON API 数据接口

## 📊 JSON API

访问 `http://localhost:8080/data` 获取：

```json
{
    "temperature": 25.3,
    "humidity": 60.5,
    "heatindex": 26.1,
    "time": "2026-03-13 14:30:25",
    "mode": "simulator"
}
```

## 🔧 对比 ESP32 版本

| 功能 | PC 模拟器 | ESP32 实物 |
|------|----------|-----------|
| 传感器 | 软件模拟 | DHT22 真实读取 |
| 显示 | Tkinter GUI | OLED 屏幕 |
| Web 服务器 | Python | ESP32 WebServer |
| 网络 | 本地/局域网 | WiFi 连接 |
| 成本 | 0 元 | 需要硬件 |

## 🎯 用途

1. **开发测试** - 无需开发板即可测试代码逻辑
2. **演示展示** - 向他人展示项目功能
3. **原型验证** - 快速验证想法
4. **学习 ESP32** - 理解工作原理

## ⚠️ 注意事项

- 传感器数据是模拟的，会有随机波动
- Web 服务器默认端口 8080
- 需要 Python 3.x 和 tkinter
