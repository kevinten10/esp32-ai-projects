## 3. ESP32-S3 AI 机器视觉开发板详解

### 3.1 核心规格

| 参数 | 规格 |
|------|------|
| **核心芯片** | ESP32-S3 (乐鑫官方) |
| **处理器** | Xtensa® 32 位 LX7 双核，240 MHz |
| **Flash** | 4MB / 8MB (可选) |
| **PSRAM** | 支持外部 PSRAM |
| **Wi-Fi** | 2.4 GHz 802.11 b/g/n |
| **蓝牙** | Bluetooth 5 (LE) |
| **GPIO** | 45 个可编程引脚 |
| **工作电压** | 3.3V |
| **USB 接口** | USB OTG (Native USB) |

### 3.2 硬件特性

#### 板载外设
| 外设 | 规格 | 接口 |
|------|------|------|
| **摄像头模块** | OV2640, 200 万像素 | DVP 接口 |
| **OLED/LCD 接口** | 支持 IIC OLED 和 SPI LCD | I2C / SPI |
| **USB 接口** | Type-C, 支持下载和通信 | USB OTG |
| **MicroSD 卡槽** | 支持 FAT/exFAT 文件系统 | SDMMC |
| **按键** | BOOT 键、RESET 键 | GPIO 控制 |
| **LED 指示灯** | 电源指示灯、GPIO 指示灯 | - |

#### AI 机器视觉功能
- 人脸检测与识别
- 猫脸/宠物识别
- 颜色识别
- 形状识别
- 二维码/条形码识别
- 物体追踪

### 3.3 引脚资源

**ESP32-S3 芯片 GPIO 分布**:
- GPIO 0 ~ GPIO21
- GPIO 26 ~ GPIO48
- 共 **45 个** 物理 GPIO 管脚

**电容触摸输入**: 14 个 GPIO 可配置为电容触摸输入

**主要外设接口**:
| 接口类型 | 数量 | 说明 |
|----------|------|------|
| SPI | 多组 | 支持 LCD、Flash 等 |
| I2C | 多组 | 支持传感器、OLED 等 |
| I2S | 多组 | 音频输入输出 |
| UART | 多组 | 串口通信 |
| ADC | 20 通道 | 12 位精度 |
| DAC | ❌ | ESP32-S3 移除了 DAC |
| TWAI (CAN) | ✓ | 支持 CAN 总线 |
| USB OTG | ✓ | 全速 USB 2.0 |

### 3.4 与 ESP32 的对比

| 特性 | ESP32 | ESP32-S3 |
|------|-------|----------|
| 处理器 | 双核 LX6 | 双核 LX7 |
| 主频 | 240 MHz | 240 MHz |
| SRAM | 较少 | 512 KB TCM |
| GPIO | 34 个 | 45 个 |
| AI 加速 | ❌ | ✓ (向量指令) |
| USB OTG | ❌ | ✓ |
| 电容触摸 | 较少 | 14 通道 |
| DAC | ✓ | ❌ (移除) |
| 功耗 | 标准 | 更低 (ULP 协处理器) |

## 4. 开发环境支持

普中 ESP32 开发板支持 **三种主流开发框架**:

### 4.1 MicroPython

**特点**:
- 适合初学者，Python 语法简单易学
- 交互式开发，快速原型验证
- 丰富的库支持

**开发工具**:
- **Thonny IDE** (推荐)
- **VSCode** + MicroPython 插件
- **uPyCraft**

**环境配置步骤**:
1. 下载 MicroPython 固件
2. 使用 esptool.py 烧录固件
3. 使用 Thonny 连接开发板
4. 开始编写 Python 代码

### 4.2 Arduino

**特点**:
- 生态丰富，库资源多
- 语法简单，适合快速开发
- 社区活跃，示例代码多

**开发工具**:
- **Arduino IDE** (官方)
- **VSCode** + PlatformIO (推荐)

**环境配置步骤**:
1. 安装 Arduino IDE 或 PlatformIO
2. 添加 ESP32 开发板支持
3. 选择普中 ESP32 开发板 (esp32dev)
4. 安装依赖库 (WiFi, OLED, 传感器等)
5. 编写代码并上传

### 4.3 ESP-IDF (乐鑫官方框架)

**特点**:
- 乐鑫官方支持，功能最全
- 适合专业开发
- 支持 FreeRTOS

**开发工具**:
- **VSCode** + ESP-IDF 插件
- **命令行工具**

**环境配置步骤**:
1. 安装 ESP-IDF 工具链
2. 配置环境变量
3. 创建新项目
4. 编译并烧录

### 4.4 其他开发方式

| 方式 | 说明 |
|------|------|
| **Mixly (米思齐)** | 图形化编程，适合教学 |
| **ESPHome** | 智能家居专用框架 |
| **Tasmota** | 开源固件，即刷即用 |

## 📝 总结

### 普中 ESP32 开发板优势

1. **性价比高**: 89 元左右，适合入门学习
2. **资料完整**: 提供 MicroPython/Arduino/ESP-IDF 三种教程

       ↓
专业 → ESP-IDF (功能最全)
