## 1. ESP32-S3 芯片规格

### 1.1 核心处理器

| 参数 | 规格 |
|------|------|
| **CPU 架构** | Xtensa® 32 位 LX7 双核处理器 |
| **主频** | 80 MHz / 160 MHz / 240 MHz (可配置) |
| **协处理器** | 超低功耗协处理器 (ULP) |
| **指令集扩展** | 向量指令 (Vector Instructions) - AI 加速 |

### 1.2 存储系统

| 参数 | 规格 |
|------|------|
| **内置 SRAM** | 512 KB (TCM - Tightly Coupled Memory) |
| **内置 ROM** | 384 KB (存放 Bootloader 和固件) |
| **外部 Flash** | 支持 Quad/Octal SPI Flash，最高 32MB |
| **外部 PSRAM** | 支持 Quad/Octal SPI PSRAM，最高 32MB |
| **缓存** | 可配置的数据缓存和指令缓存 |

### 1.3 无线通信

| 参数 | 规格 |
|------|------|
| **Wi-Fi 标准** | IEEE 802.11 b/g/n (2.4 GHz) |
| **Wi-Fi 带宽** | 20 MHz / 40 MHz |
| **Wi-Fi 速率** | 最高 150 Mbps (HT40) |
| **蓝牙版本** | Bluetooth 5 (LE) |
| **蓝牙 Mesh** | 支持 |
| **远距离模式** | 支持 Coded PHY (125 kbps / 500 kbps) |
| **发射功率** | -20 ~ 20.5 dBm (可配置) |
| **接收灵敏度** | -98 dBm (BLE, 1 Mbps) |

### 1.4 安全特性

| 安全功能 | 说明 |
|----------|------|
| **安全启动** | 基于 RSA-3072 算法，验证固件签名 |
| **Flash 加密** | 基于 AES-XTS-256 算法 |
| **数字签名** | 支持 RSA/ECC 签名引擎 |
| **HMAC 模块** | 硬件 HMAC 加速器 |
| **世界控制器** | 提供两个隔离的执行环境 |
| **随机数生成器** | 硬件真随机数发生器 |

## 2. 普中 ESP32-S3 开发板硬件

### 2.1 开发板整体规格

| 参数 | 规格 |
|------|------|
| **产品型号** | 普中-ESP32S3 AI 机器视觉开发板 |
| **核心模块** | ESP32-S3-WROOM-1 / N16R8 |
| **Flash 容量** | 16MB (N16) |
| **PSRAM 容量** | 8MB (R8) |
| **USB 接口** | USB Type-C (原生 USB OTG) |
| **工作电压** | 3.3V (内部 LDO 稳压) |
| **输入电压** | 5V (USB 或 VIN 引脚) |
| **工作电流** | 平均 ~100mA (Wi-Fi 传输时峰值 ~350mA) |
| **工作温度** | -40°C ~ +85°C (工业级) |

### 2.2 板载外设

| 外设 | 型号/规格 | 接口 | 用途 |
|------|-----------|------|------|
| **摄像头接口** | 24-pin FPC 插座 | DVP | OV2640/OV5640 摄像头 |
| **显示屏接口** | 40-pin FPC 插座 | SPI/8080/RGB | LCD/OLED 显示屏 |
| **MicroSD 卡槽** | 支持 SD/SDHC/SDXC | SDMMC/SPI | 数据存储 |
| **USB Type-C** | USB 2.0 Full Speed | USB OTG | 下载/通信/供电 |
| **按键** | 2 个轻触开关 | GPIO | BOOT/RESET |
| **LED 指示灯** | 2 个 LED | GPIO/电源 | 电源/状态指示 |
| **蜂鸣器** | 有源/无源可选 | GPIO | 声音提示 |
| **光敏电阻** | 光敏传感器 | ADC | 环境光检测 |
| **RGB LED** | WS2812 或类似 | RMT | 状态指示 |

### 2.3 扩展接口

| 接口类型 | 数量 | 排针规格 |
|----------|------|----------|
| **GPIO 排针** | 多组 | 2.54mm 间距 |
| **I2C 接口** | 专用排针 | SDA/SCL/GND/VCC |
| **SPI 接口** | 专用排针 | MOSI/MISO/SCK/CS |
| **UART 接口** | 专用排针 | TX/RX/GND |
| **ADC 接口** | 多通道 | 3.3V 兼容 |

## 3. 引脚定义与功能

### 3.1 ESP32-S3 GPIO 总览

ESP32-S3 共有 **45 个物理 GPIO 管脚**:
- **GPIO 0 ~ GPIO 21** (22 个)
- **GPIO 26 ~ GPIO 48** (23 个)

> ⚠️ **注意**: GPIO 22 ~ GPIO 25 不存在

### 3.2 普中 ESP32-S3 开发板引脚分配

#### 电源引脚

| 引脚 | 名称 | 类型 | 说明 |
|------|------|------|------|
| 1 | 3V3 | P | 3.3V 电源输出 (最大 500mA) |
| 2 | GND | P | 地 |
| 3 | VIN | P | 5V 电源输入 |

#### 启动和调试引脚

| 引脚 | 名称 | 类型 | 说明 |
|------|------|------|------|
| 4 | EN | I | 使能引脚 (高电平有效) |
| 5 | BOOT0 | I | 启动模式选择 (下拉=运行模式，上拉=下载模式) |

### 3.3 默认外设引脚分配 (普中开发板)

#### I2C 接口
| 信号 | GPIO | 说明 |
|------|------|------|
| SDA | GPIO 21 | I2C 数据线 |
| SCL | GPIO 22 | I2C 时钟线 (⚠️ 注意：ESP32-S3 无 GPIO22，实际使用 GPIO 48) |

> ⚠️ **重要**: ESP32-S3 没有 GPIO 22-25，普中开发板可能重新映射了 I2C 引脚

#### SPI 接口 (VSPI)
| 信号 | GPIO | 说明 |
|------|------|------|
| SCK | GPIO 18 | 时钟 |
| MISO | GPIO 19 | 主入从出 |
| MOSI | GPIO 23 | 主出从入 (⚠️ ESP32-S3 无 GPIO23，实际使用 GPIO 47) |
| CS | GPIO 5 | 片选 |

#### UART 接口
| 接口 | TX | RX | 说明 |
|------|-----|-----|------|
| UART0 | GPIO 1 | GPIO 3 | 调试串口 |
| UART1 | GPIO 43 | GPIO 44 | 用户可用 |
| UART2 | GPIO 16 | GPIO 17 | 用户可用 |

#### ADC 通道
| ADC 单元 | 通道 | GPIO |
|----------|------|------|
| ADC1 | CH0-CH12 | GPIO 32-44 |
| ADC2 | CH0-CH14 | GPIO 20, 26-31, 45-48 |

## 4. 外设接口详解

### 4.1 摄像头接口 (DVP)

| 信号 | GPIO | 说明 |
|------|------|------|
| PCLK | GPIO 40 | 像素时钟 |
| VSYNC | GPIO 41 | 垂直同步 |
| HREF | GPIO 42 | 水平参考 |
| XCLK | GPIO 39 | 外部时钟 |
| D0-D7 | GPIO 43-48 | 数据位 0-7 |
| SIOC | GPIO 47 | SCCB 时钟 |
| SIOD | GPIO 48 | SCCB 数据 |
| PWDN | GPIO 38 | 电源控制 |
| RESET | GPIO 46 | 复位 |

**支持的摄像头传感器**:
- OV2640 (200 万像素)
- OV5640 (500 万像素)
- GC0308 (30 万像素)

### 4.2 显示屏接口

#### SPI LCD 引脚
| 信号 | GPIO | 说明 |
|------|------|------|
| CS | GPIO 5 | 片选 |
| DC | GPIO 6 | 数据/命令选择 |
| MOSI | GPIO 7 | 数据输出 |
| SCK | GPIO 8 | 时钟 |
| RESET | GPIO 9 | 复位 |
| BLK | GPIO 10 | 背光控制 |

#### RGB LCD 引脚 (8080 接口)
| 信号 | GPIO | 说明 |
|------|------|------|
| HSYNC | GPIO 47 | 行同步 |
| VSYNC | GPIO 48 | 场同步 |
| DE | GPIO 45 | 数据使能 |
| PCLK | GPIO 46 | 像素时钟 |
| D0-D15 | GPIO 37-44 | 数据位 0-15 |

### 4.3 SD 卡接口

| 信号 | GPIO | 说明 |
|------|------|------|
| CMD | GPIO 21 | 命令 |
| CLK | GPIO 22 | 时钟 |
| D0 | GPIO 23 | 数据位 0 |
| D1 | GPIO 24 | 数据位 1 |
| D2 | GPIO 25 | 数据位 2 |
| D3 | GPIO 26 | 数据位 3/片选 |

> ⚠️ **注意**: ESP32-S3 无 GPIO 22-25，SD 卡引脚需重新映射

### 4.4 USB OTG

ESP32-S3 支持 **原生 USB OTG** 功能:
- **USB D+**: GPIO 19
- **USB D-**: GPIO 20

可用于:
- USB 串口 (CDC)
- USB MIDI 设备
- USB 键盘/鼠标
- USB 存储设备

## 📝 附录

### A. 引脚使用注意事项

1. **GPIO 6-11**: 连接内部 Flash，不建议作为普通 GPIO 使用
2. **GPIO 22-25**: ESP32-S3 没有这些引脚
3. **ADC2 通道**: 使用 Wi-Fi 时不可用
4. **GPIO 34-39**: 仅支持输入模式 (无内部上拉)
5. **启动引脚**: GPIO 0 决定启动模式 (0=下载，1=运行)

### B. 推荐 GPIO 分配

| 用途 | 推荐 GPIO | 理由 |
|------|-----------|------|
| **数字输入** | GPIO 4, 5, 12-21 | 通用，无冲突 |
| **数字输出** | GPIO 4, 5, 12-21 | 通用，无冲突 |
| **ADC 测量** | GPIO 32-44 (ADC1) | 不受 Wi-Fi 影响 |
| **I2C** | GPIO 47, 48 | 可配置，无冲突 |
| **SPI** | GPIO 5, 18, 19, 47 | 硬件 SPI 支持 |
| **UART** | GPIO 43, 44 | 硬件 UART1 |
| **触摸** | GPIO 32-44 | 支持电容触摸 |
